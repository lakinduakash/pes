# pes  - Project Evaluation System
Automate project evolution system for universities

### To Initialise server module
Run following 
```
    cd server
    npm install
```
#### Start sever

```
    node app.js
```

### To Initialise ui module
Run following 
```$bash
    cd ui
    npm install
```
#### run project

```$bash
    ng serve
```
Module will start at `localhost:4200`
